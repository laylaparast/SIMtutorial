# Surrogate Marker Evaluation Tutorial


Download the .Rmd file, open in Rstudio, and Knit to HTML. Alternatively, go [here](https://htmlpreview.github.io/?https://github.com/laylaparast/SIMtutorial/blob/main/Parast_SIM_tutorial.html) to view the HTML version of this R markdown directly. The tutorial follows the paper: "Surrogate Marker Evaluation: A Tutorial Using R" by Layla Parast, in Statistics in Medicine (2025),
44(10-12):e70048, which is available in this repository (Parast2025_Tutorial.pdf).

The tutorial covers the following packages: `Rsurrogate`, `hetsurr`, `OptimalSurrogate`, and `SurrogateParadoxTest`. You can find more information about this paper and tutorial [here](https://github.com/laylaparast/SIMtutorial), the `Rsurrogate` package more generally [here](https://github.com/laylaparast/Rsurrogate), the `hetsurr` package more generally [here](https://github.com/laylaparast/hetsurr), the `OptimalSurrogate` package more generally [here](https://github.com/celehs/OptimalSurrogate), and the `SurrogateParadoxTest` package more generally [here](https://github.com/emily13hsiao/SurrogateParadoxTest).

You can find more information on software for surrogates [HERE](https://www.surrogatemarkers.com/software).
